yajl-py by pykler

Pure Python wrapper to the Yajl C Library

This is a python wrapper using the python standard library ctypes. Yajl provides a way of doing stream parsing (sax-like for lack of a better term) parsing of JSON by firing callbacks whenever a JSON unit is parsed. The main advantage of using Yajl from python using Yajl-py is to be able to process parts of the JSON document rather than having to place the entire parsed JSON document in memory as is the traditional usecase with the builtin python json library. The usecase for using Yajl-py + Yajl over the builtin python json library for JSON is the same usecase for using SAX over Minidom when parsing XML.

Dependencies

Python (2.6, not tested with 3+ series) Yajl (http://lloyd.github.com/yajl/)

Install

Clone this repo: git clone git://github.com/pykler/yajl-py.git Install yajl-py: easy_install yajl-py
